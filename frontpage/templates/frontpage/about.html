About

Ghost Artemis is a socially motivated group that creates tools and provides information for protecting yourself.

Ghost Artemis went from being a toy to a project to becoming an small organization. With electronics often being a central part of our lives, we want to build a platform for a better future.

I want to leave behind a better world. If I were to be reborn, I would want to have built a tree that provides the shade. Cybersecurity (and ultimately computer science) will pave that path for us.

Security does so by allowing us to live in peace and comfort. It allows us to think and live in dignity. We can only do this when we are allowed to be different. Come build this world with me!
